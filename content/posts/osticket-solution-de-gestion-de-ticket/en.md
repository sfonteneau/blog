I was looking for a ticketing / incident management solution that is powerful but simple. I tried GLPI, but it didn’t fit what I wanted.

I wanted a solution where customers don’t need an account to submit a ticket: someone outside the organization should be able to request support easily.

Requests and replies can be handled via email or the web interface. Personally, I prefer disabling the email workflow: the web interface is so well done that it’s the best option.

osTicket features:
- Manage ticket states (open/closed/resolved)
- Auto‑responses
- SLA management
- Ticket submission forms
- …
